## Usage Guidelines

1. **Lead with the verb** - Start bullets with action
2. **Match intensity to level** - Senior roles = stronger verbs
3. **Vary verbs** - Don't repeat same verb 3+ times
4. **Pair with metrics** - Strong verb + number = impact
5. **Context matters** - Use verbs that fit the industry
