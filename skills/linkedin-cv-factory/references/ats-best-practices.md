# ATS Best Practices Reference

## Format Requirements

### ✅ ATS-Friendly Elements
- Standard fonts: Arial, Calibri, Garamond, Georgia
- Section headers: EXPERIENCE, EDUCATION, SKILLS (all caps)
- Simple bullets: • or -
- Left-aligned text
- Consistent spacing
- Single column layout

### ❌ ATS Problem Elements
- Tables or text boxes
- Headers/footers with contact info
- Images or graphics
- Multiple columns
- Fancy symbols or icons
- PDF forms with fields

## Keyword Placement Hierarchy

1. **Professional Summary** (40% of keyword weight)
   - Front-load critical skills
   - Include job title variations
   - Mention industry expertise

2. **Core Competencies/Skills Section** (30% of keyword weight)
   - Bullet list of 8-12 key skills
   - Match exact job description terms
   - Include both acronyms and full terms

3. **Experience Bullets** (20% of keyword weight)
   - Weave keywords naturally
   - Use in context of accomplishments
   - Include relevant tools/platforms

4. **Education/Certifications** (10% of keyword weight)
   - List relevant certifications
   - Include degree keywords if specified

## Keyword Types to Include

### Hard Skills
- Technical tools and platforms
- Programming languages
- Software names
- Methodologies (Agile, Scrum, etc.)
- Industry-specific terms

### Soft Skills (Use Sparingly)
- Leadership
- Communication
- Problem-solving
- Collaboration

### Contextual Keywords
- Industry buzzwords
- Company-specific terms
- Job function synonyms
- Regional terminology

## Scoring Formula

```
ATS Score = (Keyword Match × 0.50) + 
            (Format Compliance × 0.30) + 
            (Seniority Alignment × 0.20)
```

**Target: >90% for competitive positions**
