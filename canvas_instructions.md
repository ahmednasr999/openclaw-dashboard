# Ahmed Nasr LinkedIn Post - Canva Design Instructions

## Canvas Setup
- **Dimensions:** 1200 × 627 px (LinkedIn optimal)
- **Background:** Gradient from #0f172a (navy) to #1e293b (dark slate)

## Design Elements

### 1. Background
- Create a gradient background: Top-left #0f172a → Bottom-right #1e293b
- Add subtle grid pattern overlay (optional):
  - Create 40px × 40px grid lines
  - Color: rgba(59, 130, 246, 0.03) - very subtle blue tint
  - Opacity: 3%

### 2. Accent Line (Top)
- Add a 4px horizontal bar at the very top
- Gradient: #3b82f6 → #ef4444 → #3b82f6
- Stretch across full width

### 3. Logo Section (Top-Left)
- **Logo Image:** Upload https://ahmednasr999.github.io/openclaw-dashboard/logo_ahmed_nasr.jpg
- **Size:** 48 × 48 px
- **Corner Radius:** 8px
- **Shadow:** 0px 4px 15px rgba(0,0,0,0.3)
- **Text:** "AHMED NASR"
  - Font: Inter (or similar sans-serif like Montserrat Bold)
  - Size: 18px
  - Color: #ffffff
  - Weight: 700
  - Letter-spacing: 0.5px
- **Position:** Logo at top-left, text to its right with 12px gap

### 4. Main Content Area (Center-Left)

#### Timestamp Banner
- **Text:** "⚠️ 2:47 AM — SYSTEM LOCKUP"
- **Font:** Inter Bold
- **Size:** 22px
- **Color:** #ef4444 (red)
- **Text Transform:** UPPERCASE
- **Letter-spacing:** 1px
- **Position:** Below logo section, 35px margin

#### Headline

- **Font:** Inter Extra Bold (800)
- **Size:** 42px

- **Line-height:** 1.15
- **Letter-spacing:** -0.5px
- **Highlight:** "30,000" in #3b82f6 (accent blue)

#### Quote Block
